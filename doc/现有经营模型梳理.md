# 现有经营模型梳理

这是一份内部资产梳理文档，记录了当前应用于信贷业务的经营模型清单，包括模型名称、应用方式、模型key、底表及原负责人。共梳理了10个模型，覆盖首借、复借、轻资产等场景，部分模型处于空跑或未应用状态。

## 关键信息

- 模型总数：10个
- 已应用模型：5个（复借G卡、首借G卡V1、首借睡眠发起意愿模型、首借睡眠交易通过模型、轻资产发起意愿模型、轻资产交易通过模型）  -- 注意：实际上已应用有6个
- 空跑/未应用模型：2个（首借G卡V2修正版、轻资产动支模型）
- 状态未明确模型：2个（复借复拨G卡、轻资产U卡）
- 数据底表依赖：8张

## 用途

该文档作为经营模型资产库的索引源，可衍生多个模型实体页面、人员页面和底表页面。
