# 0610 早晨修改版文档发送说明

本目录可脱离项目单独发送，包含三份 Markdown 文档和功能介绍中引用的真实页面截图。

## 文件清单

- `项目功能介绍-0610早晨修改版.md`
- `用户使用流程-0610早晨修改版.md`
- `项目架构介绍-0610早晨修改版.md`

## 使用说明

请保持 `assets/` 目录和三份 Markdown 文档在同一层级下，这样 `项目功能介绍-0610早晨修改版.md` 中的截图可以正常显示。

文档中的 Mermaid 架构图需要在支持 Mermaid 的 Markdown 查看器中渲染；如果查看器不支持 Mermaid，正文内容仍可正常阅读。
